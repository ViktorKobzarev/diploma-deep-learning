# diploma-deep-learning
# Применение глубокого обучения для распознавания и классификации изображений

Магистерская диссертация, 2026

## Аннотация

Цель данной работы — повысить точность распознавания объектов на изображении, используя ансамблевый метод **стекинг**, по сравнению с отдельными нейронными сетями.

## Датасет

В работе использован датасет **NIH ChestX-ray14**, содержащий 112 120 изображений рентгеновских снимков грудной клетки с 14 патологиями (15-й класс — отсутствие заболеваний). 

Особенности датасета:
- Сильный дисбаланс классов
- Задача мультилейбл классификации (один снимок может содержать несколько патологий)

## Методология

В ансамбль (стекинг) включены три предобученные легковесные модели (выбраны из-за ограниченности вычислительных ресурсов):

| Модель | Тип |
|--------|-----|
| DenseNet-121 | CNN |
| EfficientNetV2-S | CNN |
| Swin-Tiny | Transformer (механизм внимания) |

**Принцип работы:**
1. Три модели обучаются независимо на частично различных данных
2. Каждая модель имеет свою функцию потерь и гиперпараметры
3. Собираются сырые логиты (предсказания) от каждой модели
4. Логиты конкатенируются и подаются на вход мета-модели (полносвязная нейросеть)
5. Мета-модель выдаёт финальный ответ

**Цель дифференцированного обучения:**
- Одна модель — лучше распознавать малые классы
- Вторая модель — оптимальное общее обучение
- Третья (Swin-Tiny) — экспериментально, через механизм внимания
